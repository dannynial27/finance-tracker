# Finance Tracker

A simple and intuitive personal finance tracking Android application.

## Features
* **Track Expenses and Income**: Keep an eye on where your money goes.
* **Categorization**: Easily categorize your transactions.
* **Analytics**: View simple charts and analytics of your spending.
* **Modern UI**: Clean and aesthetically pleasing user interface built with Jetpack Compose.

## Run Locally

**Prerequisites:** [Android Studio](https://developer.android.com/studio)

1. Open Android Studio
2. Select **Open** and choose the directory containing this project
3. Let Android Studio sync the Gradle project
4. Run the app on an emulator or a physical Android device

## Architecture
This app is built using modern Android development practices, including Jetpack Compose, ViewModels, and Room Database for local data storage.
